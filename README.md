package skyBlueProject;

public class mensagensAutomaticas {
	String nome;
    double preco;

    public void Produto(String nome, double preco) {
        this.nome = nome;
        this.preco = preco;
    }

    public void exibirProduto() {
        System.out.println("Produto: " + nome);
        System.out.println("Preço: R$ " + preco);
    }
}

O codigo acima é referente ao parte do programa que vamos criar para solucionar o problema da empresa Skyblue e logo abaixo vou falar das principais diferenças dos softwares IDE..

As IDEs são ferramentas usadas para desenvolver softwares mas nem todas sao iguais, elas possuem diferenças importantes entre si.
Algumas sao mais leves, simples e tem uma interface mais chamativa e bonita assim como o Vs code e  enquanto outras são mais completas e pesadas, como o Eclipse e o IntelliJ.
Mas a principal diferença está na linguagem de programação suportada. Algumas IDEs são específicas para certas linguagens, como o PyCharm para Python, enquanto outras suportam várias linguagens.
Outra diferença entre elas é a quantidade de recursos e ferramentas que algumas possuem e tambem nao podemos esquecer da interface e facilidade no uso.. algumas sao mais "secas" vamos assim dizer tendo apeno basico que se usa.
